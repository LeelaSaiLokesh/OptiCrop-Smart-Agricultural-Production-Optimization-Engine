Project Overview

What is OptiCrop?

A data-driven, AI-powered solution reimagining how farmers, researchers, and agribusinesses make crop decisions.

OptiCrop is an intelligent crop recommendation engine that leverages Machine Learning and agricultural data science to recommend the most suitable crop based on real-world soil and climate parameters.

Farmers traditionally rely on experience, intuition, or manual advisory services — an approach that is slow, inconsistent, and error-prone. OptiCrop transforms this process into a data-driven, instant, and accurate prediction experience powered by an ensemble of 5 validated ML algorithms.

The system accepts 7 agronomic parameters — Nitrogen (N), Phosphorus (P), Potassium (K), Temperature, Humidity, pH, and Rainfall — and returns the optimal crop recommendation with a 99.5% accuracy confidence backed by Random Forest.

ML Lifecycle

Machine Learning Workflow

A rigorous, 9-stage end-to-end machine learning pipeline — from raw dataset to production-ready prediction API.

📦
Step 01 · Data Acquisition

Dataset Collection

Curated Kaggle Crop Recommendation Dataset with 2,200 records across 22 crop classes. Features: N, P, K, Temperature, Humidity, pH, Rainfall — all with known agronomic significance.

🧹
Step 02 · Data Quality

Data Cleaning

Missing value imputation, duplicate removal, outlier detection using IQR method, and data type validation to ensure a clean, consistent feature matrix for modelling.

📊
Step 03 · Analysis

Exploratory Data Analysis

Statistical summaries, correlation heatmaps, distribution plots (Matplotlib/Seaborn), class balance analysis, and feature-target relationship exploration to guide engineering decisions.

⚙️
Step 04 · Feature Engineering

Feature Engineering

Created engineered features: NPK Ratio, Soil-Climate Interaction Index, humidity-temperature correlation composite. Applied StandardScaler normalization for algorithm convergence.

🤖
Step 05 · Model Development

Model Training

Trained 5 classification algorithms: Random Forest (300 estimators), Decision Tree (max_depth=15), KNN (k=5), SVM (RBF kernel), Logistic Regression (lbfgs solver) on 80/20 train-test split.

📋
Step 06 · Performance Analysis

Model Evaluation

Comprehensive evaluation with Accuracy, Precision, Recall, F1-Score, ROC-AUC, Confusion Matrix, and Classification Report. 10-fold stratified cross-validation for statistical reliability.

🔧
Step 07 · Optimization

Hyperparameter Optimization

GridSearchCV and RandomizedSearchCV applied to all 5 models. Tuned n_estimators, max_depth, min_samples_split, learning_rate, and kernel parameters for peak performance.

🏆
Step 08 · Selection

Model Selection

Random Forest selected as production model: 99.5% accuracy, superior F1-Score across all 22 classes, best ROC-AUC, and robust generalization on unseen data.

🚀
Step 09 · Deployment

Prediction Pipeline

Flask REST API serializes the trained model as a .pkl file and serves real-time predictions. Input → Preprocessing → Inference → Postprocessing → JSON Response to frontend.

FAQ

Frequently Asked Questions

Answers to common questions from evaluators, recruiters, and users about OptiCrop.

OptiCrop is an AI-powered Smart Agricultural Production Optimization Engine — a SmartBridge capstone project that uses Machine Learning to recommend the most suitable crop for a given set of soil and climate conditions. It accepts 7 agronomic parameters (N, P, K, Temperature, Humidity, pH, Rainfall) and returns the optimal crop recommendation with a confidence score. The system is built with Python, Flask, Scikit-Learn, and Bootstrap 5 for a fully responsive web experience.

The prediction workflow follows these steps: (1) User inputs soil and climate parameters through the web form. (2) Flask backend validates and preprocesses inputs using StandardScaler normalization. (3) The pre-trained Random Forest classifier (300 estimators, GridSearchCV optimized) processes the feature array. (4) The model generates a 22-class probability distribution. (5) The highest-probability class is returned as the recommended crop with its confidence score. The model achieves 99.5% accuracy validated through 10-fold stratified cross-validation on the Kaggle crop dataset.

OptiCrop is designed for farmers, agricultural researchers, agribusiness professionals, government agricultural departments, and students. The required inputs are: Nitrogen (0–140 ppm), Phosphorus (5–145 ppm), Potassium (5–205 ppm), Temperature (8–44 °C), Humidity (14–100 %), pH (3.5–10), and Rainfall (20–300 mm). Soil test values can be obtained from soil testing laboratories, government soil health cards, or portable soil test kits available from agricultural cooperatives.

ML Algorithms: Random Forest (best — 99.5%), Decision Tree (97.7%), K-Nearest Neighbors (97.5%), Support Vector Machine (96.8%), Logistic Regression (90.2%), K-Means Clustering (EDA).

Backend: Python 3.11, Flask 3.0, Pickle model serialization, Jinja2 templating.

ML Stack: Scikit-Learn 1.3, Pandas 2.0, NumPy 1.26, Matplotlib, Seaborn, Jupyter Notebook.

Frontend: HTML5, CSS3, Bootstrap 5.3, Bootstrap Icons, Vanilla JavaScript, Google Fonts.

Dev: Git, GitHub, VS Code, Virtualenv.

The production model is the Random Forest Classifier with 300 estimators, achieving 99.5% accuracy on the held-out test set. It was selected after GridSearchCV hyperparameter optimization and 10-fold stratified cross-validation across all 5 candidate algorithms. The model achieves an F1-Score of 0.995 across all 22 crop classes, with a macro-averaged ROC-AUC score exceeding 0.999. All evaluation metrics are documented in the project's Jupyter Notebook.

The complete OptiCrop project — including source code, Jupyter Notebooks, trained models, dataset, and documentation — is available on GitHub. The repository is organized with the following structure: app/ (Flask application), models/ (trained .pkl files), dataset/ (training data), reports/ (evaluation metrics and visualizations), and scripts/ (training scripts). This project is part of the SmartBridge AI/ML Internship capstone programme.
