Machine Intelligence

The ML Engine Behind OptiCrop

A rigorous, production-ready machine learning pipeline designed for agricultural intelligence, model interpretability, and predictive accuracy.

ML Pipeline Architecture

Data Collection & Preprocessing
Kaggle crop recommendation dataset (2200 records) with label encoding, missing value imputation, and outlier detection using IQR method.
Feature Engineering
Soil-Climate Interaction Index, NPK ratio analysis, humidity-temperature correlation, and StandardScaler normalization for optimal model convergence.
Model Training & Selection
5 algorithms evaluated: Random Forest, Decision Tree, KNN, SVM, and Logistic Regression. GridSearchCV hyperparameter tuning with 10-fold stratified cross-validation.
Model Evaluation
Comprehensive evaluation using Accuracy, Precision, Recall, F1-Score, ROC-AUC, and Confusion Matrix. Comparative analysis across all 5 algorithms.
Prediction Pipeline
Flask REST API serving pickle-serialized models with real-time preprocessing, standardization, and multi-class probability output for the crop recommendation.
